# AgroSense

Projeto desenvolvido para monitorar a umidade do solo com Arduino e sensor capacitivo. A solução realiza a leitura contínua dos dados, converte os valores em porcentagem e apresenta uma visualização mais clara para apoiar decisões de irrigação.

## O que tem neste repositório até o momento:
- Código da lógica utilizada no Arduino
- Lógica da calculadora financeira
- Script do banco de dados

## Tecnologias até o momento:
Arduino IDE, Arduino, sensor capacitivo de umidade do solo, GitHub, HTML & JavaScript e MySQL.
